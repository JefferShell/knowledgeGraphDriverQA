## 项目介绍

**医疗知识图谱展示系统** 是一个基于 neo4j 的医疗知识图谱展示系统。该系统可展示和搜索医疗知识图谱中的节点和关系。

### 主要功能

* 展示医疗知识图谱的节点和关系。
* 利用三元组查询功能搜索相关知识。
* 支持问答功能（需要手动填充问答数据）。

### 技术架构

* **前端:** HTML、CSS、JS
* **后端:** Django
* **数据库:** SQLite
* **图谱引擎:** Neo4j
* **图谱展示:** ECharts

### 使用说明

1. 启动项目并登录系统。
2. 探索知识图谱展示页面。
3. 利用三元组查询功能搜索相关知识。
